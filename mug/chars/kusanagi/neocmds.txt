 *** M.U.G.E.N. JOYSTICK CONFIGURATION ***
 (Player 1)

 *- DIRECTION KEYS

		     (U)           U = up        DF = downforward
    
 		 (B)  x  (F)       B = back      DB = downback
  
  		     (D)           F = Forward   UF = upforward

		                   D = down      UB = upback

 *- BUTTONS

		 (x) (y) (z)       x = light punch    y = hard punch   z = No Use 
                   
		 (a) (b) (c)       a = light kick     b = hard kick    c = No Use

		     (s)                              s = start
